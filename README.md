# hahow-video-downloader

一键下载好学校hahow.in课程视频

## Usage

` Node.js 版本: 14.x`

* change `config.json.sample` to `config.json` and fill in the configuration
* `brew install yarn` or `npm install --global yarn`
* `yarn`
* run `node index`
